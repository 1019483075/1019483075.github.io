1.router-link :

  1. 简单的定死路由跳转：:路由跳转，将组件映射到路由。

  1. 动态路由：就是登陆页面除了名字不同其他是相同的跳转的是同一个组件。假设一个 user 组件，不同用户(用户的 id)会导航到同一个 user 组件中。就是 path 属性不能写死。

2.redirect 重定向

  1. 概念：是指当页面初始化进入事指向的那个路由。
