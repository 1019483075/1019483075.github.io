1.字符串函数

  1. 字符串函数
    1. 字符串的输出
      1. echo：输出字符串
      2. var_dump()：输出值和类型以及长度
      3. print_r()：输出值
      4. die()；退出当前状态并且输出一条信息
      5. exit()；与 die 的用法类似
      6. printf();格式化输出
      7. sprintf()；与 printf（）类似
  2. 字符串的处理介绍

    1. 去除空格和字符串填充

      1. trim(),ltrim(),rtrim();分别是去掉字符串左右空格，去掉字符串左边空格，去掉字符串右边空格。
      2. strlen()：获取字符串长度，包含空格。
      3. str_pad()：给字符串填充，默认是往右边填充 str_pad(填充字符，填充长度，填充使用的字符串默认空白，规定填充字符串那边);
      4. str_repeat()：指定字符串重复几遍 str_repeat(string,次数)；

    2. 字符串大小写转换

      1. strtolower()将字符转换为小写

      2. strtoupper()将字符转换为大写

      3. ucfirst()首字母单词大写

      4. unwords()每个单词字母大写

    3. 与 html 标签和关联的字符串函数

      1. nl2bar()：2 是指 to 转换的意思吧 nl 转换为 br

      2. trip_tags()：是指将 html 标签去掉，只留下文字

      3. htmllspecialchars()：标签被显示出来不是被解析

      4. addslashes()：

      5. stripslashes()

    4. 其他字符串格式化函数

      1. strrev()：语句反转

      2. strlen()：获取字符串长度

      3. number_format()：货币格式化

      4. md5()：加工成密文 加工后的密文是 32 位的密文

    5. 字符串分割与拼接

      1. explode()：吧字符串打撒为数组

      2. implode()：吧数组元素组合为字符

      3. join():是指 implode()的别名。

    6. 字符串截取

      1. substr()：对字符串进行截取，定义开始于结束。截取的最后一位不会包括

      2. mb_substr():对多个字节进行处理。不会出现乱码现象

    7. 字符串查找

      1. strpos()：查找字符串最开始出现的位置，从索引值 0 开始查找

      2. strrpos()：查找字符串最后出现的位置

    8. 字符串替换

      1. str_replace()：用一个字符提哈另外一个字符

    9. 其他字符串函数

      1. pathinfo()：地址解析成一个数组

      2. basename()：获取路径的文件名

      3. dirname():获取路径名

      4. parse_url()：获取地址下的协议 路径 以字符串的形式返回

      5. parse_str()：吧地址参数切割组装成一个数组
