  2. 系统数组函数

    1. 数组键值操作函数

      1. array_values()获取数组的值

      2. array_keys()获取数组的键

      3. in_array()检查一个值是否在数组中

      4. array_key_exists()检查一个键是否在数组中

      5. array_flip()数组键值对调

      6. array_reverse()数组索引值与值对调

    2. 统计数组的元素和唯一性

      1. count()统计数组元素的个数

      2. array_count_values()统计数组值出现的次数

      3. array_unique()统计数组的唯一值

    3. 使用回调函数处理数组和函数

      1. array_filter()过滤数组的值

      2. array_map()将回调函数改成数组的值

    4. 数组排序函数

      1. sort()不保留原数组 key 升序排序

      2. rsort()不保留原数组 key 降序排序

      3. asort()保留原来的 key 升序排序

      4. arsort()保留原来的 key 降序排序

      5. ksort()按键排序 升序 保留 key

      6. krsort()按键排序 降序 保留 key

      7. natsort()按自然数排序

      8. natcasesort()忽略大小写排序

    5. 拆分，合并，分解与结合函数

      1. array_slice():根据数组索引值截取指定长度。array_slice(array,start,length,preserve)

      2. array_splice()删除并替换数组中的元素

      3. array_combine() 数组合并

      4. explode()数组拆分

      5. implode()数组合并

      6. join()是 implode()的别名

    6. 其他有用的数组处理函数

      1. range()生产有规律的数组
      2. array_sum()计算数组之和
      3. shuffle()打乱数组
      4. array_rand()随机取 key

    7. 数组与数据结构

      1. array_pop()删除数组最后一个元素

      2. array_shift()删除数组最开始的一个元素

      3. array_push()数组后面插入一个值 返回数组长度

      4. array_unshift()数组前面插入一个值 返回数组的长度

3.print_r()与 var_dump()的区别：

  1. print_r()能打印复杂数组类型的值，可以打印数组结构和内容，按照一定格式显示键和元素

  2. var_dump() 判断一个变量的类型的数值，如果变量有值，输出的是变量的数值，返回数据类型
