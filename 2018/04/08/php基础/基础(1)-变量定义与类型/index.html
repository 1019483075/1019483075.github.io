mysql，apache，php 之间的关系:
apache:是一个服务器，是构筑一个 php 和数据库可以沟通的环境。是指 php 和 mysql 两个不同语言的人打电话，通过 Apache 这个同声翻译系统进行构建。

1.php 定义

  1. php 的定义 php 以<?php 开始,代码块 以 ?>结束 php 的拓展名是.php
  2. php 的变量以$符号开头，后面跟变量名规则：必须以字母或者是下划线字符开始。变量名只能包含数字字符及下划线（A-Z,0-9 和_）。不能包含空格。严格区分大小写。
  3. echo 和 print 输出的区别 echo：可以输出一个或者是多个字符。
  4. print：只允许输出一个字符，echo 比 print 输出块，echo 没有返回值，print 有返回值 1。

2.php 变量类型

  1. 整数（$a=10）

  2. 浮点($a=10.10)

  3. 字符型($a=”一串字符”)

  4. 布尔型($a=true)

  5. 数组($a=array())

  6. 对象(由一切事物组成，包括行为，特征)；

  7. 资源（操作数据库资源）

  8. null()
    对象理解

  9. *public*是指对属性(特征)或者是方法(行为)的控制，public（公有的），protected（）

  10. ->,=>,::,class 的使用：*->*是指实例化的对象，读取类中的属性和方法。=>
    用于数组"key"=>"value",    ::静态方法和动态方法法引用。class 声明一个类。

  11. 代码如下：
    class Person{
    //属性是静态的
    public name="liuliu";
    public age=18;
    public sex="nan";
    //方法是动态的
    public function say(){
    echo "今天的天气真是好";
    }
    }
    //实例化一个对象
    $user1=new Person;
    echo $user1->name;

数据库资源理解
注意：数据库必须在数据库连接成功的情况下进行操作。

  1. 连接数据库：$connet=mysql_connet(servername,username,passworld,dbname,port);

  2. 定义 msql 语言:mysql_query($connet,”set name usf8”);

  3. 选择 msql 数据库语言:mysql_select_db(“数据库名”,”$connet”);

  4. 准备查询语句:$sql=”select * from user”;

  5. 吧语句发送给数据库:$rst=mysql_query($sql,$connet);

  6. 从 msql 返回结果
    $arr=array();
    while($row=mysql_fetch_array(rst)){
    var_dump($row);
    arr_push($arr,$row);

    }

7.关闭数据库:mysql_close($connet);

3.字符串与字符串的连接

$a=”xxx”;
$b=”xxx”;
echo $a.$b;

4.单引号与双引号的区别

  1. 双引号里面可以解析变量，但是要以${变量名}的格式。
  2. 但引号不可以解析变量。
  3. 双引号里面可以解析变量但是不可以解析表达式。但是可以以”the num id”.($a=$b).”!!!!”;
  4. 双引号中解析变量加双引号操作:echo “my name is"${a}"!!!”;

变量测试与转换

1.测试变量是否存在

  1. gettype() 获取变量的类型 类似于 var_dump()
  2. isset() 判断变量是否设置,是否为 null
    $aa=array('test'=>1,'hello'=>NULL);
    echo var_dump(isset($aa['test']);//true
  3. empty() 判断一个变量是否为空

2.变量类型之间的转换

  1. 自动转换
    1. int，bool,float,string,array,object
    2. 例如：$num1=3.14; $num2=(int)$num1;
  2. 强制转换
    1. intval(),floatval(),strval()
    2. 例如：$str=”123.9abc”; $int=intval($str);
    3. 通用类型转换 settype() 例如:$num4=12.8; $flg=settype($num4,”int”);
