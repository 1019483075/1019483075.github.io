1.数组的方法

  • slice（start，end（不包含））：从已有的数组返回指定的元素。改方法不会修改数组，而是返回一个子数组。
    • start：必须。规定从何处开始选取。如果是负数，那么规定从数组尾部算起，也就是说-1 是指最后一个元素。
    • end：可选。规定从何处结束选取。如果没有指定该参数，那么切分的数组包含从 start 数组结束的元素。如果该参数是负数从尾部算起。
  • splice（index，howmany,item1,…itemxx）：向数组添加或者是删除项目。
    • index：必须。整数。添加或者是删除数组的位置，使用负数可以从数组尾部算起。
    • howmany：必须。要删除项目的数量。如果为 0 就是不删除。
    • item：可选。向数组添加新项目。
  • concat():创建并返回一个新的数组。新数组由原数组的 concat()里面的参数组成。原数组不变。
  • sort():以字母表顺序排列
  • reverse():将数组中的元素倒序顺序，返回逆序的字符串
  • join():拼接数组中的元素并且返回新的字符串，不改变原数组。
